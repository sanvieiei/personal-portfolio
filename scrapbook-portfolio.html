Project #1:
Pyrate's Fuel Run
project: Pyrate's Fuel Run
✿ Pyrates Fuel Run
What does it do? High-energy 2D arcade adventure built in pygame, where you captain a pirate through choppy seas, dodge surprise obstacles, and race to make it home.
Pyrates Fuel Run

What I’m proud of

As you dive deeper into the voyage, the whole scene transforms with a smooth color shift so every checkpoint feels like a fresh new world.

Tech stack

Python + pygame, with custom game loop tuning, procedural obstacle spawning, and dynamic audio layering.

Anecdote

Fueled by late-night energy, I kept building past midnight with my friend on the call—iterating features mid-chat and turning rough ideas into one of my favorite jam sessions.

Tech feel: powerful visuals with simple interactivity.
→ github repo
Project #2:
ParentPal
project: ParentPal
✿ ParentPal
What does it do? AI platform where parents can upload photos of their child to get a general diagnosis of any skin issues.
ParentPal

What I’m proud of

We were able to teach our model how to interpret images of healthy baby skin and unhealthy baby skin, and the model showed correct results!

Tech stack

Python, React.js, Gemini API, Flask, SQLAlchemy.

Anecdote

Created this project at my first ever hackathon: TerraHacks 2025! We may not have won but it was so fun!

Design note: reading is easy and not cluttered, with soft color separation.
→ youtube demo
